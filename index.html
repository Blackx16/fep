Unit-1: Introduction to Scripting

What is Scripting?

  • A scripting language or script language is a programming language that supports the writing of scripts.
  • Scripts automate the execution of tasks which could alternatively be executed one by one by a human operator.
  • Scripting languages are usually interpreted from source code.

Client-side Scripting / Server-side Scripting

  • Client-side: Executed and changed by the user on the client side (browser).
  • Server-side: Executed on the server; user cannot change or disable it.

How the Web Works

  • JavaScript code runs inside a web page loaded into a browser.
  • Web pages can be loaded directly from the hard drive or from web servers using HTTP.

Introduction to JavaScript

  • JavaScript allows the creation of interactive web pages.
  • It is an interpreted language, running in the browser to add dynamic content like scrolling text, menus, pop-up windows, animations.

Applications/Uses of JavaScript

  1. Data Entry Validation: Prequalifies data entered by the user.
  2. Dynamic HTML Interactivity: Creates dynamic content like image swaps, pop-up windows.
  3. Offloading a Busy Server: Converts frequently used processes to client-side scripts.
  4. Adding Life to Dead Pages: Adds interactivity to engage users.
  5. Creating Web Pages That Think: Makes pages appear “smart” by remembering previous visits.
  6. Browser Detection: Detects browser type and loads specific content.
  7. Pop-up Windows: Displays content in smaller windows.
  8. Dynamic Content: Shows content based on user preferences.

Advantages of JavaScript

  1. Interpreted Language: No compilation step required.
  2. Embedded Within HTML: No special editor needed; can be written in any text editor.
  3. Minimal Syntax: Easy to learn and use.
  4. Quick Development: No time-consuming compilation; quick testing and debugging.
  5. Platform Independence: Works on any JavaScript-enabled browser.
  6. Event-based Programming: Supports object/event-based programming.

Using JavaScript on a Web Page

  • JavaScript is embedded in HTML using the <script> tag.
  • Scripts can be placed in <head> or <body> sections of the HTML document.

Unit-2: Basics of JavaScript

User Interaction Through Dialog Boxes

  • Alert Dialog Box: Displays a message with an OK button. 
     
alert("Your message here");
  • Prompt Dialog Box: Displays a message with an input field. 
    
var name = prompt("Enter your name", "Default value");
alert(name);

Built-in Functions

  1. parseInt(): Converts a string to an integer. 
    
var num = parseInt("123");
  2. parseFloat(): Converts a string to a floating-point number. 
    
var num = parseFloat("123.45");
  3. Number(): Converts a string to a number. 
    
var num = Number("123.45");
  4. isNaN(): Checks if a value is NaN. 
    
if (isNaN(value)) {
    alert("Not a number");
}

Flow Control Statements

  • if Statement: Executes code block if condition is true. 
    
if (condition) {
    // code block
}
  • if-else Statement: Executes code block based on condition. 
    
if (condition) {
    // code if true
} else {
    // code if false
}
  • else-if Statement: Checks multiple conditions. 
    
if (condition1) {
    // code
} else if (condition2) {
    // code
} else {
    // code
}
  • switch Statement: Checks value against multiple cases. 
    
switch (expression) {
    case value1:
        // code
        break;
    case value2:
        // code
        break;
    default:
        // code
}

Looping Statements

  • for Loop: Repeats code a specified number of times. 
    
for (var i = 0; i < 5; i++) {
    // code block
}

Arrays

  • Arrays store a sequence of values.
  • Declaring an array: 
    
var arr = new Array(size);
var arr = [];
  • Initializing a dense array: 
    
var arr = [10, 20, "Hello"];

User-defined Functions

  • Functions group code into a single reusable unit. 
    
function functionName(param1, param2) {
    // code block
}

Parameter Passing

  • Values passed to functions: 
    
functionName(value1, value2);

Function Call

  • Calling a function: 
    
functionName(parameters);

Returning a Value

  • Returning a value from a function: 
    
function functionName(param) {
    return param * 2;
}
var result = functionName(10);

Highlights

  • JavaScript is an interpreted language used for creating interactive web pages.
  • Client-side scripts run in the user's browser, server-side scripts run on the server.
  • JavaScript can validate data, add dynamic content, and reduce server load.
  • Key concepts include dialog boxes, built-in functions, control statements, loops, arrays, and functions.

Unit-3: Working with Objects

Objects, Properties, and Methods

  • JavaScript is an object-oriented language.
  • Objects: Entities like tables, chairs, cars, etc., defined by their properties and methods.

Objects in JavaScript

  • Objects help reduce programming efforts.
  • Using JavaScript Objects: Each object has properties and methods to manipulate data.
  • Creating an Object: Use the var keyword to define a variable initialized with an object.
var var_name = new ObjectName(optional_parameters);

Example:

var a = new Array(5);
var b = new Date();
var c = new String("Hello");

Using an Object’s Properties

  • Access values using variable.propertyName.
var_name.propertyName

Example:

var a = new Array("Hello", 123, 1);
alert(a.length); // OUTPUT: 3

Calling an Object’s Methods

  • Methods perform tasks, similar to functions.
var_name.methodName

Example:

var a = new Date();
alert(a.getFullYear()); // OUTPUT: 2011

Primitives and Objects

  • Variables store data directly; objects reference memory addresses.
  • JavaScript automatically converts data types as needed.

JavaScript Native Objects

  • Native Objects: String, Array, Math, Number, Date

String Object

  • String values and objects allow the same methods.
  • Create a string object using new String().
var variable_name = new String(Some string or value);

Example:

var a = new String("Hello");
var b = new String(123);
alert(a); // OUTPUT: Hello
alert(b); // OUTPUT: 123

Property of String Object

  • length: Returns the number of characters in the string (read-only).
object_variable.length

Example:

var a = new String("Hello"), b = "FYBCA";
alert(a.length); // OUTPUT: 5
alert(b.length); // OUTPUT: 5

Methods of String Object

  1. toLowerCase(): Converts a string to lowercase.
some_var = string_var.toLowerCase();

Example:

var a = new String("Hello"), b = "Fybca";
alert(a.toLowerCase()); // OUTPUT: hello
alert(b.toLowerCase()); // OUTPUT: fybca
  1. toUpperCase(): Converts a string to uppercase.
some_var = string_var.toUpperCase();

Example:

var a = new String("Hello"), b = "Fybca";
alert(a.toUpperCase()); // OUTPUT: HELLO
alert(b.toUpperCase()); // OUTPUT: FYBCA
  1. charAt(): Extracts a single character from a string.
some_var = string_var.charAt(index_position);

Example:

var a = new String("Hello"), b = "Fybca";
alert(a.charAt(0)); // OUTPUT: H
alert(b.charAt(b.length - 1)); // OUTPUT: a
  1. indexOf(): Searches for a substring within a string.
some_var = string_var.indexOf(index_position, start_search_position);

Example:

var a = new String("Hello");
alert(a.indexOf("e")); // OUTPUT: 1
alert(a.indexOf("e", 2)); // OUTPUT: -1
alert(a.indexOf("E")); // OUTPUT: -1
  1. substr(): Extracts a substring from a string.
some_var = string_var.substr(start_position, number_of_characters);

Example:

var a = new String("Hello");
alert(a.substr(0, 2)); // OUTPUT: He
alert(a.substr(3, 2)); // OUTPUT: lo

Methods of Math Object

  1. min(): Finds the minimum value among the provided parameters.

alert(Math.min(a, b, c)); // OUTPUT: 9.5

  1. setFullYear(): Changes the year number.

Regular Expressions

  • A sequence of characters forming a search pattern.
  • Used for text search and replace operations.

What is a Regular Expression?

  • Describes what you are searching for in a text.
  • Can be a single character or a complex pattern.
/pattern/modifiers

Example:

var pattern = /GSFCU/i;
  • /GSFCU/i is a regular expression.
    • GSFCU is the pattern.
    • i is a modifier for case-insensitivity.

Using String Methods with Regular Expressions

  • search(): Searches a string for a specified value, returns the position of the match.
var str = "This is GSFCU";
var n = str.search("GSFCU"); // OUTPUT: 8

Example:

var str = "This is GSFCU";
var n = str.search(/GSFCU/i); // OUTPUT: 8
  • replace(): Replaces a specified value with another value in a string.
var str = "Visit Microsoft!";
var res = str.replace("Microsoft", "GSFCU");

Example:

var str = "Visit Microsoft!";
var res = str.replace(/microsoft/i, "GSFCU"); // OUTPUT: Visit GSFCU!

Regular Expression Modifiers

  • i: Case-insensitive search.

Unit-4: Basics of DOM

Introduction to DOM

  • An HTML page is rendered in a browser, which assembles all elements in its memory.
  • Post-rendering, browsers cannot recognize individual HTML elements.
  • JavaScript-enabled browsers use the DOM (Document Object Model) to recognize individual HTML objects after rendering.
  • The DOM allows browsers to access and manipulate object properties anytime.
  • JavaScript object-based browsers create JavaScript objects mapping to the DOM.

DOM Hierarchy

  • Available in all scriptable browsers.
  • Core DOM available in all browsers; HTML DOM hierarchy varies by browser and HTML elements.
  • Forms a tree-like structure with the root object at the top (window object).
  • Accessing a deeply nested object requires traversing from the root through intermediate objects.
  • Standard DOM objects have methods for access/manipulation and properties to modify application of methods.

Form DOM Hierarchy

  • HTML documents can contain objects like Images, Hyperlinks, Frames, Forms, etc.
  • Browser creates arrays in memory for HTML objects (e.g., Images, Links, Forms).
  • Form elements must be named and can be accessed by their names in JavaScript.
  • Form elements can be accessed as objects with properties like name, value, type.
  • getElementById collection is a W3C standard, supported by all browsers.
// Syntax for accessing form elements
window.document.formName.elementName.property/method/event
window.document.forms[i].elementName.property/method/event
window.document.forms["Name"].elementName.property/method/event

Example:

a = window.document.frm.txt1.value;
b = window.document.forms[0].txt1.value;
c = window.document.forms["frm"].txt1.value;

Accessing Elements Data

  • User input is captured in a Form, creating a forms array in memory.
  • Elements in a form are accessed through the forms collection.
  • Each element is referenced using its name followed by the property/method/event.

Example:

var a;
a = window.document.forms[0].txt1.value;
alert(a);
window.document.forms[0].txt1.value = "FYBCA";

Event Handling

  • A web page event is associated with user actions (e.g., mouse clicks, key presses).
  • JavaScript's approach involves identifying a web page object, choosing an event, and connecting it to JavaScript code snippets.

Types of Events

  • Interactive Events: Depend on user interaction (e.g., onMouseOver).
  • Non-Interactive Events: Do not need user interaction (e.g., onLoad).

Handling Events in JavaScript (Event Handlers)

  • Binding events through tag attributes: Assign inline JavaScript code in the attribute of an element.
<input type="button" name="btn" value="Click here" onClick="abc();" />

Events in DOM

  1. Click event: onClick fires when an object is clicked.
<input ... onClick="functionName();" />
  1. Blur event: onBlur fires when the cursor moves away from an object.
<input ... onBlur="functionName();" />
  1. Focus event: onFocus fires when the cursor enters an object.
<input ... onFocus="functionName();" />
  1. Load event: onLoad fires after the page/document is loaded.
<body onLoad="functionName();" />
  1. MouseDown event: onMouseDown fires when an element is clicked and held.
<input ... onMouseDown="functionName();" />
  1. MouseUp event: onMouseUp fires when a mouse click is released.
<input ... onMouseUp="functionName();" />
  1. MouseOut event: onMouseOut fires when the cursor leaves an element.
<input ... onMouseOut="functionName();" />
  1. MouseOver event: onMouseOver fires when the cursor hovers over an element.
<input ... onMouseOver="functionName();" />
  1. Reset event: onReset fires when the form’s reset button is pressed.
<form ... onReset="functionName();" />
  1. Submit event: onSubmit fires when the form’s submit button is pressed.
<form ... onSubmit="functionName();" />

Unit-5: DOM Programming

Document Object

  • Each loaded document becomes a document object.
  • Access HTML elements, properties, and methods through this object.
window.document.propertyName
window.document.methodName([parameter])
  • Browser creates arrays in memory per HTML object in the document.

Referencing an Element

  • The document object is the owner of all objects in a web page.
  • Access elements by id, tag name, or class name.

Finding HTML Elements by ID

var element = document.getElementById("idname");

Example:

<p id="a">some text here…</p>
var paragraph = document.getElementById("a");

Finding HTML Elements by Tag Name

var elements = document.getElementsByTagName("tagname");

Example:

<p>some text here…</p>
<p>another paragraph</p>
var paragraphs = document.getElementsByTagName("p");

Finding HTML Elements by Class Name

var elements = document.getElementsByClassName("classname");

Example:

<p class="a">some text here…</p>
<h1 class="a">Some heading here…</h1>
var elements = document.getElementsByClassName("a");

Modifying HTML Element’s Content

  • Use properties like innerHTML, innerText, and textContent.

innerHTML

  • Sets or returns the HTML content of an element.
element.innerHTML = somevalue;
var content = element.innerHTML;

Example:

<p id="a">something <u>here</u></p>
<p id="b"></p>
var para1 = document.getElementById("a");
var para2 = document.getElementById("b");
para2.innerHTML = para1.innerHTML;

innerText

  • Returns just the text content of the element without extra spacing and all its children.
element.innerText = somevalue;
var text = element.innerText;

Example:

<p id="a">something <u>here</u></p>
<p id="b"></p>
var para1 = document.getElementById("a");
var para2 = document.getElementById("b");
para2.innerText = para1.innerHTML;

textContent

  • Returns the text content of the element and all descendants, with extra spacing but without tags.
element.textContent = somevalue;
var text = element.textContent;

Example:

<p id="a">something <u>here</u></p>
<p id="b"></p>
var para1 = document.getElementById("a");
var para2 = document.getElementById("b");
para2.textContent = para1.textContent;

Adding HTML Elements Dynamically

  • Use methods like createTextNode, createElement, appendChild, and append.

createTextNode()

  • Creates a text node.
var textNode = document.createTextNode("text to be added as node");

Example:

<p>some paragraph</p>
var text = document.createTextNode("I was added after the paragraph");
document.body.appendChild(text);

createElement()

  • Creates an element node.
var element = document.createElement("elementType");

Example:

var h1 = document.createElement("h1");
var text = document.createTextNode("New Heading");
h1.appendChild(text);
document.body.appendChild(h1);

appendChild()

  • Appends a node as the last child of an element.
parentElement.appendChild(childElement);

append()

  • Appends multiple nodes or strings at once.
element.append(node1, node2, ...);

Example:

var h1 = document.createElement("h1");
var text = document.createTextNode("New Heading");
h1.append(text, " hello");
document.body.appendChild(h1);

Deleting HTML Elements

  • Use methods like removeChild and remove.

removeChild()

  • Removes a specified child node from the current node.
parentElement.removeChild(childElement);

Example:

document.body.removeChild(document.getElementById("a"));

remove()

  • Removes an element from the document.
element.remove();

Example:

var element = document.getElementById("demo");
element.remove();

Adding Attributes to HTML Elements Dynamically

  • Use setAttribute to add or update attributes.
element.setAttribute("attributeName", "newValue");

Example:

var anchor = document.getElementById("myAnchor");
anchor.setAttribute("href", "https://www.gsfcuni.edu.in/");

Deleting Attributes of HTML Elements

  • Use removeAttribute to remove attributes.
element.removeAttribute("attributeName");

Example:

var anchor = document.getElementById("myAnchor");
anchor.removeAttribute("href");
